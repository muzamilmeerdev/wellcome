# 🌟 Muzamil's Personal Portfolio

Modern, futuristic portfolio website with dark mode, neon gradients, and glassmorphism effects.

## ✨ Features
- 🎨 Dark theme with blue & purple neon gradients
- ✨ Smooth animations with Motion (Framer Motion)
- 📱 Fully responsive design
- 🔗 Direct WhatsApp, Instagram & Call integration

## 🚀 Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- Motion (Framer Motion)
- Lucide Icons
